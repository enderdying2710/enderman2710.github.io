二、综合评估：你适合的方向

根据你的技能体系、项目经验和学历背景，推荐优先级如下：

第一选择：Java后端开发 / 全栈开发（最匹配）

你的核心竞争力在 Spring Boot + MongoDB 后端开发，且有完整的全栈项目经验。市场上对 Java 后端的需求量大，薪资天花板高，且你已有实际项目的架构设计能力（三层架构、索引策略、JWT认证等），这是你最应该优先投递的方向。

第二选择：软件测试（高匹配度）

你有80+条测试用例的实战经验，掌握黑盒测试、接口测试、兼容性测试全流程，这是一个非常有说服力的数据。测试岗位对学历门槛通常比开发低，作为大专生进入中大厂的"曲线救国"路径非常可行。但长期来看薪资天花板比开发低。

第三选择：微信小程序开发（差异化竞争）

你精通 uni-app + Vue3 + uview-plus，有完整的微信小程序全栈开发经验。很多中小公司/创业公司急需能独立完成小程序开发的工程师，你的全栈能力在这里是稀缺竞争力。

⭐⭐⭐⭐⭐ 极力推荐（技能+学历高度匹配）

2 广州昊誉信息科技有限公司 AI方向后端/全栈开发工程师 10-18K 技能完美匹配（Spring Boot+全栈），要求「熟练使用AI编程助手（Claude Code、Cursor）」正好是你的技术栈加分项。虽然写着2年经验，但你校鱼项目的深度不输1-2年工作经验，值得冲。

四、关于三种简历方向的建议

Java开发简历 vs 软件开发简历 vs 软件测试简历

你准备了三份简历，说明你对不同方向都有准备。我的建议是：

方向 适合投递的场景 投递策略
Java开发简历 后端开发岗、Java工程师岗 主打简历，投递70%的岗位
软件开发简历（全栈） 全栈工程师、小程序开发、初创公司 投递20%的岗位，突出独立交付能力
软件测试简历 测试工程师、QA岗 投递10%的岗位作为保底方案

简历优化建议

  1. 学历不占优，用项目补：校鱼项目深度足够（7个MongoDB集合+20+ API+全流程），简历中要突出"独立设计"、“从需求到交付"等关键词，让面试官关注你的实际能力而非学历。

  2. 量化成果：

    • 开发方向：强调"开发20+ RESTful API”、“设计7个MongoDB集合”、“全页面真实API对接”
    • 测试方向：强调"编写80+条测试用例"、“发现setData大文件白屏等关键缺陷”

  3. 差异化定位：你的优势是「懂开发 + 会测试 + 能做全栈」，这在应届生中是稀缺组合。建议在简历中体现这个交叉优势。

五、求职策略总结

策略 具体行动
海投策略 大枣科技（技术储备）、昊誉信息（全栈开发）、自我游（Java开发）这三个优先投
简历侧重 以 Java开发/软件开发简历为主力，软件测试简历为保底
薪资预期 大专应届生第一份工作预期 5-8K 比较合理，技术特别对口的可以要到 8-12K
长期规划 先入行积累1-2年经验 → 跳槽到更大平台 → 大专学历不再成为瓶颈
技能提升 持续关注AI编程（Claude Code/Cursor）、大模型应用开发，这是未来1-2年的技术红利期

六、一句话总结

你的技术实力（Spring Boot全栈 + 完整项目）远超一般大专应届生，建议主攻Java后端开发方向，把昊誉信息的「AI方向全栈开发工程师」作为第一目标，大枣科技的「技术储备专员」作为稳妥保底。大专不是天花板，你的项目深度才是。

本报告基于34家企业的招聘数据和个人简历信息分析生成，仅供参考。
